Каждая задача считается выполненной только при выполнении ВСЕХ условий:

### Для кода:
- [ ] Код написан и соответствует принципам проекта
- [ ] Покрыт unit-тестами (минимум 80%)
- [ ] Проходит code review (если команда >1)
- [ ] Документация обновлена
- [ ] Все тесты проходят (`pytest tests/ -v`)

### Для документации:
- [ ] Документ создан/обновлён
- [ ] Проверен на орфографию и ясность
- [ ] Добавлен в оглавление

- [ ] Протестировано на реальных данных (файлы 1С)
- [ ] Производительность соответствует требованиям
