## 📊 Приоритеты модуля прогнозирования

| Приоритет | Функция | Бизнес-ценность |
|-----------|---------|-----------------|
| **P0** | Авто-выбор модели | Пользователь не думает о методах |
| **P0** | Доверительные интервалы | Показывает неопределённость |
| **P0** | Визуализация прогноза | Наглядность для директора |
| **P1** | Сезонная декомпозиция | Понимание паттернов |
| **P1** | Прогноз для всех метрик | Универсальность |
| **P2** | Объяснение прогноза (LLM) | Доверие к результату |
| **P2** | Сценарии «Что если» | Планирование |
| **P3** | Прогнозирование иерархий | Согласованность уровней |
