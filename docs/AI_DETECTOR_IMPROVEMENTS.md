
### 2. Расширенные эвристики (Level 2)

#### Динамический расчет confidence
```python
# Раньше: фиксированный confidence (0.75)
# Теперь: динамический на основе данных

confidence = 0.70 + (matched_count / 10) * 0.25  # 0.70 - 0.95
```

#### Новые типы эвристик
Добавлено 7 новых типов эвристик:
1. `_is_cost_column()` - себестоимость/затраты
2. `_is_profit_column()` - прибыль/маржа
3. `_is_price_column()` - цена
4. `_is_category_column()` - категория/группа
5. `_is_manager_column()` - менеджер/ответственный
6. `_is_region_column()` - регион/город
7. `_is_document_column()` - документ/номер

#### Умная обработка веса и объема
```python
# Для веса/объема допускаем дробные числа
"ВесНетто": [100.5, 200.3, 300.7] → quantity (0.77)
"ОбъемМ3": [1.5, 2.0, 3.5] → quantity (0.77)

# Для штук - только целые числа
"КолвоШтук": [5, 10, 15] → quantity (0.82)
```

#### Контекстный анализ значений
```python
# Для строк проверяем содержание
customer_keywords + string_values → confidence на основе count
string_count >= 3 → confidence 0.65 - 0.95
```

## 📝 Рекомендации по использованию

### Для лучших результатов:
1. **Именуйте колонки понятно** - используйте стандартные термины 1С
2. **Избегайте аббревиатур** - пишите полные названия
3. **Проверяйте confidence < 0.7** - такие колонки требуют ручной проверки
4. **Используйте metadata** - там хранится информация о найденных ключевых словах

### Интеграция:
```python
from src.core.parser import ExcelParser
from src.core.ai_detector import AIDetector

parser = ExcelParser()
detector = AIDetector()

# Загрузка файла
df = parser.load_file("data.xlsx")
df = parser.clean_data(df)

# Распознавание структуры
mappings = detector.detect_columns(
    df.columns.tolist(), 
    df.to_dict(orient='list')
)

# Анализ результатов
summary = detector.get_mapping_summary(mappings)
print(f"Высокая уверенность: {summary['high_confidence']}")
print(f"Требуют проверки: {len(summary['requires_review'])}")
```
