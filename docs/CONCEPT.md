# Концепция продукта: 1C Dashboard Service

## Проблема
- 1.5+ млн компаний в РФ используют 1С
- 68% МСБ недовольны отчётностью в 1С
- BI-интеграции дорогие (от ₽100 000) и сложные
- Пользователи выгружают данные в Excel и работают вручную

## Решение
SaaS-сервис, который:
1. Принимает сырой XLSX из 1С
2. AI распознаёт структуру (колонки, иерархии, типы данных)
3. Применяет бизнес-правила для расчёта метрик
4. Генерирует дашборд с подходящими визуализациями
5. Экспортирует в PNG/PDF/PPTX
6. **Прогнозирует будущие значения ключевых показателей**

## Ключевые принципы

1. **Data-driven** — система предлагает только то, что можно построить из данных
2. **No overload** — максимум 12 компонентов на главный экран
3. **Progressive disclosure** — сначала главное, детали — по клику
4. **Learn from corrections** — сохраняет исправления пользователя
5. **Industry-aware** — разные правила для розницы, опта, производства
6. **Local-first AI** — Ollama для конфиденциальности, облако — опция
7. **Smart Forecasting** — автоматический выбор лучшей модели прогноза

## Модуль прогнозирования

### Поддерживаемые методы
| Метод | Когда используется | Точность | Скорость |
|-------|-------------------|----------|----------|
| **Prophet** | Есть сезонность, праздники | Высокая | Средняя |
| **ARIMA/SARIMA** | Стационарные ряды, тренды | Высокая | Низкая |
| **Exponential Smoothing** | Короткие ряды, простой тренд | Средняя | Высокая |
| **XGBoost/LightGBM** | Много факторов, нелинейность | Очень высокая | Средняя |
| **Neural Networks (LSTM)** | Длинные ряды, сложные паттерны | Очень высокая | Низкая |
| **Ensemble** | Автоматическая комбинация моделей | Максимальная | Низкая |

### Авто-выбор модели
```python
def select_best_model(time_series):
    checks = {
        'length': len(time_series),
        'seasonality': detect_seasonality(time_series),
        'trend': detect_trend(time_series),
        'stationarity': adf_test(time_series),
        'outliers': count_outliers(time_series)
    }
    
    if checks['length'] < 30:
        return 'exponential_smoothing'
    elif checks['seasonality'] and checks['length'] > 100:
        return 'prophet'
    elif checks['stationarity']:
        return 'arima'
    else:
        return 'xgboost'
```

### Визуализация прогнозов
- Линейный график с доверительным интервалом (80%, 95%)
- Тепловая карта сезонности
- Декомпозиция ряда (тренд + сезонность + остаток)
- Сравнение прогноза с фактом (MAPE, MAE, RMSE)
