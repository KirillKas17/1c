## 5. Требования к модулю прогнозирования

### 5.1 Методы
| Метод | Библиотека | Когда используется |
|-------|-----------|-------------------|
| Exponential Smoothing | statsmodels | Короткие ряды, простой тренд |
| ARIMA/SARIMA | statsmodels | Стационарные ряды, сезонность |
| Prophet | prophet | Сезонность, праздники, выбросы |
| XGBoost/LightGBM | xgboost/lightgbm | Много факторов, нелинейность |
| Neural Networks (LSTM) | tensorflow/pytorch | Длинные ряды, сложные паттерны |
| Ensemble | custom | Автоматическая комбинация |

### 5.2 Метрики качества
| Метрика | Допустимое значение |
|---------|--------------------|
| MAPE | < 15% (краткосрок), < 25% (долгосрок) |
| Coverage (95% ДИ) | 90-98% |
| Bias | Близко к 0 |

### 5.3 Производительность
| Параметр | Требование |
|----------|-----------|
| Подбор модели | < 2 секунд |
| Построение прогноза (100 точек) | < 5 секунд |
| Память на модель | < 500 MB |

### 5.4 Визуализация
- Линейный график: факт + прогноз + доверительный интервал
- Декомпозиция: тренд + сезонность + остаток
- Тепловая карта сезонности
- Таблица метрик качества
