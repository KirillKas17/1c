# Мобильная адаптация веб-интерфейса

## Обзор реализации

Полная мобильная адаптация веб-интерфейса парсера документов для устройств с шириной экрана до 768px.

## Реализованные функции

### 1. Адаптивное меню навигации

**Мобильное гамбургер-меню:**
- Кнопка меню появляется только на мобильных устройствах (≤768px)
- Фиксированное позиционание в левом верхнем углу
- Плавная анимация открытия/закрытия
- Затемнение фона при открытом меню (overlay)

**Боковая панель (Sidebar):**
- На десктопе: фиксированная ширина 280px
- На мобильных: скрыта по умолчанию, выезжает слева при открытии
- На мобильных показывает только иконки без текста
- Автоматическое закрытие после выбора пункта меню
- Скроллинг контента при переполнении

### 2. Адаптивная типографика

**Заголовки:**
- H1: 2xl → 1.5rem на мобильных
- H2: 3xl → 1.25rem на мобильных  
- H3: xl → 1.1rem на мобильных

**Отступы:**
- p-8 → 1rem
- p-6 → 0.75rem
- p-4 → 0.5rem
- p-3 → 0.5rem

### 3. Адаптивные кнопки

- Уменьшенные размеры: padding 10px 16px
- Размер шрифта: 0.875rem
- Минимальная высота тач-элементов: 44px (стандарт accessibility)
- `touch-action: manipulation` для улучшения отклика

### 4. Адаптивная статистика

**Карточки статистики (#history-stats):**
- Десктоп: 4 колонки
- Планшет: 2 колонки
- Мобильный (<375px): 1 колонка
- Уменьшенный размер шрифта чисел: 2xl → 1.5rem

### 5. Адаптивные фильтры

**Пресеты фильтров:**
- Автоматическое обтекание (flex-wrap)
- Кнопки занимают 50% ширины на мобильных
- На очень маленьких экранах (<375px): 100% ширины

**Сетка фильтров:**
- Все поля в одну колонку на мобильных
- Сохранение полной функциональности

### 6. Адаптивная таблица истории

**Горизонтальный скролл:**
- Контейнер таблицы с `overflow-x: auto`
- `-webkit-overflow-scrolling: touch` для плавности на iOS
- Минимальная ширина таблицы: 800px

**Скрытие колонок:**
- Скрываются на мобильных: ИНН, Дата документа, Дата обработки
- Остаются видимыми: Имя файла, Тип, Статус, Контрагент, Сумма, Действия
- Класс `.hide-mobile` для управления видимостью

**Размеры:**
- Размер шрифта таблицы: 0.75rem
- Уменьшенные padding ячеек: 0.5rem
- `white-space: nowrap` для предотвращения переноса

### 7. Адаптивные карточки документов

**Сетка карточек:**
- Десктоп: до 4 колонок (2xl+)
- Планшет (1024px): мин. высота 350px
- Мобильный (768px): мин. высота 300px
- Автоматическое заполнение (`auto-fit`)

### 8. Адаптивные модальные окна

- Ограничение высоты: `max-height: 90vh`
- Вертикальный скролл при переполнении
- Отступы: margin 1rem
- Предотвращение выхода за пределы экрана

### 9. Ориентация устройства

**Пейзажная ориентация на мобильных:**
- Уменьшенная ширина sidebar: 240px
- Дополнительные уменьшения заголовков
- Оптимизация использования пространства

### 10. Очень маленькие экраны (<375px)

- Дополнительное уменьшение sidebar: 260px → 240px
- Пресеты фильтров в одну колонку
- Статистика в одну колонку

## Технические детали

### CSS Breakpoints

```css
/* Планшеты и меньше */
@media (max-width: 1024px) { ... }

/* Мобильные устройства */
@media (max-width: 768px) { ... }

/* Очень маленькие экраны */
@media (max-width: 375px) { ... }

/* Пейзажная ориентация */
@media (max-width: 768px) and (orientation: landscape) { ... }
```

### JavaScript функции

**`toggleMobileMenu()`**
- Управление состоянием мобильного меню
- Переключение классов `.open` и `.active`
- Работа с overlay для затемнения фона

**Автозакрытие меню**
- Event listeners на всех элементах навигации
- Проверка ширины экрана перед закрытием
- Сохранение поведения на десктопе

**Переинициализация иконок**
- `lucide.createIcons()` после переключения вкладок
- Задержка 100ms для гарантии рендеринга

### Accessibility

- Минимальный размер тач-целей: 44×44px (WCAG 2.1)
- `touch-action: manipulation` для быстрых кнопок
- Плавные переходы для всех интерактивных элементов
- Сохранение контрастности на всех размерах
- Поддержка keyboard navigation

### Производительность

- CSS-only анимации (transform, opacity)
- Hardware acceleration через `will-change`
- Ленивая загрузка иконок
- Оптимизированные селекторы

3. **Таблица**
   - Горизонтальный скролл
   - Просмотр скрытых колонок
   - Действия с документами

4. **Ориентация**
   - Поворот устройства
   - Сохранение функциональности
   - Корректное отображение

## Отличия от десктопной версии

| Элемент | Десктоп | Мобильный |
|---------|---------|-----------|
| Sidebar | Видим всегда | Скрыт, по клику |
| Навигация | Текст + иконки | Только иконки |
| Таблица | Все колонки | 6 из 9 колонок |
| Статистика | 4 колонки | 2 колонки |
| Фильтры | 6 колонок | 1 колонка |
| Кнопки | Полноразмерные | Компактные |
| Отступы | Стандартные | Уменьшенные |

## Заключение

Мобильная адаптация обеспечивает полное сохранение функциональности десктопной версии при оптимизации под ограниченный размер экрана. Все критические функции доступны, интерфейс интуитивно понятен и соответствует современным стандартам mobile UX.
