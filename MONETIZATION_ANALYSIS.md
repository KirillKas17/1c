## 📈 ФИНАНСОВАЯ МОДЕЛЬ

### Юнит-экономика (на 1 клиента):

| Показатель | Start | Pro | Enterprise |
|------------|-------|-----|------------|
| **ARPU** (средний чек) | 990₽ | 4,900₽ | 19,900₽ |
| **CAC** (стоимость привлечения) | 1,500₽ | 5,000₽ | 25,000₽ |
| **Churn** (отток в месяц) | 8% | 5% | 2% |
| **LTV** (пожизненная ценность) | 12,375₽ | 98,000₽ | 995,000₽ |
| **LTV:CAC** | 8.25x | 19.6x | 39.8x |
| **Payback period** | 2 мес | 1 мес | 1.5 мес |

**Расчёт LTV:**
```
LTV = ARPU × (1 / Churn Rate)
Start: 990₽ × (1 / 0.08) = 12,375₽
Pro: 4,900₽ × (1 / 0.05) = 98,000₽
Enterprise: 19,900₽ × (1 / 0.02) = 995,000₽
```
